# DPU-SmartSwitch


- **DPU**, 数据处理单元（`Data Processing Unit`, DPU）是一种可编程的计算机处理器，它将通用中央处理器（CPU）与网络接口（NIC）硬件紧密集成。

- 它们偶尔也被称为“IPU”（基础设施处理单元）或“智能网卡（SmartNICs）”。

- 它们可以替代传统的网络接口卡（NICs），减轻主CPU的复杂网络职责和其他“基础设施性”任务；尽管它们的功能各不相同，但可用于执行加密/解密、充当防火墙、处理传输控制协议/网际协议（TCP/IP）、处理超文本传输协议（HTTP）请求，甚至用作虚拟机监控程序或存储控制器。

- 这些设备可能对云计算提供商具有吸引力，否则他们的服务器可能会在这些任务上花费大量CPU时间，从而减少可提供给客户的周期。与通用CPU相比，它们在常规网络任务中具有更高的功耗效率，因此也在其他类型的数据中心环境中得到应用。

- 典型能力：
  - 网络：线速处理 100G + 流量、虚拟交换（OVS）、路由、隧道、流量整形。
  - 存储：NVMe-oF、块 / 文件存储虚拟化、数据压缩 / 去重。
  - 安全：硬件加密 / 解密、防火墙、微分段、零信任。
  - 管理：带内 / 带外管理、固件升级、遥测、故障自愈。

- **SONiC**中，**SmartSwitch** 指集成 DPU 的智能数据中心交换机，是融合交换、计算、安全的一体化平台。
